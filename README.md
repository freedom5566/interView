* git指令介面無法顯示中文問題
```sh
vim .git/config
[core]下增加quotepath=false
```

* 縮排總是亂跳的問題        
這個選項是根據你打開的文件來猜測你的縮排是幾個空白，設成false
`"editor.detectIndentation":false`
